## 项目目的

本项目旨在帮助大家了解并入门VOS的相关内容

1. **光流图**：理解光流图如何表示物体运动的方向和速度。
2. **幅度图**：通过指定帧序号从视频中提取帧数据。
3. **相位图**：使用 OpenCV 的 `cv2.calcOpticalFlowFarneback` 方法计算光流并生成伪彩色光流图。
4. **伪标签**：使用 Matplotlib 展示光流图和原始帧。
