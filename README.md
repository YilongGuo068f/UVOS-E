### README: 仓库使用文档

## 项目介绍

记录我在尝试的一个过程

1. **光流图**：物体运动的方向和速度
2. **幅度图**：频率成分强度图
3. **相位图**：图像的语义特征，图像的骨架
4. **伪标签**：人的直觉判断
5. **EMA**：平滑的处理新的数据信息
6. **高斯采样**：随机生成符合高斯分布的数据
